# Нейрокомпьютерные интерфейсы - Классификация ЭЭГ данных

Проект по классификации электроэнцефалографических (ЭЭГ) данных с использованием глубокого обучения.

## Описание

Реализация алгоритма классификации ЭЭГ сигналов на основе архитектуры EEGNet. Проект включает:
- Загрузку и предобработку ЭЭГ данных из открытого датасета (MNE Sample Dataset)
- Модель CNN для классификации различных ритмов мозга
- Обучение и оценку модели
- Визуализацию результатов

**Используемый датасет**: MNE Sample Dataset (Motor Imagery) - реальные ЭЭГ данные из открытого источника

## Установка

```bash
pip install -r requirements.txt
```

## Использование

### Визуализация данных
```bash
python visualize_data.py
```

### Обучение модели
```bash
python train.py
```

## Структура проекта

- `data_loader.py` - Загрузка и предобработка данных
- `model.py` - Архитектуры моделей (EEGNet, SimpleCNN)
- `train.py` - Скрипт обучения
- `visualize_data.py` - Визуализация данных
- `doc.md` - Полная документация проекта

## Результаты

Модель достигает точности ~85-90% на тестовой выборке для классификации 4 классов ЭЭГ ритмов.

Подробная документация доступна в файле `doc.md`.
